# 音频文件说明

在此文件夹中放置音频文件，或直接使用程序内置的音效。

## 当前状态

- ✅ **提示音**：已内置回退方案（Web Audio API 生成）
- ⚠️ **背景音**：需要外部文件，或保持静音

## 文件列表

1. **notification.mp3** - 倒计时结束提示音（可选）
   - 建议时长：1-3 秒
   - 用途：番茄钟结束时播放
   - 如果不放置此文件，将使用内置音效

2. **ambient.mp3** - 工作背景音（可选）
   - 建议时长：可循环播放（1-10 分钟）
   - 用途：工作时播放的舒缓背景音
   - 建议类型：白噪音、雨声、轻音乐、自然音效等

## 免费音效资源网站

### 推荐网站

1. **Pixabay** (https://pixabay.com/sound-effects/)
   - 免费商用，无需署名
   - 推荐搜索词：
     - 提示音：`notification`, `bell`, `ding`, `soft chime`
     - 背景音：`ambient`, `rain`, `white noise`, `meditation`, `nature sounds`

2. **Mixkit** (https://mixkit.co/free-sound-effects/)
   - 免费商用，无需注册
   - 直接下载使用

3. **Freesound** (https://freesound.org/)
   - 需要注册，音效丰富
   - 注意查看每个音效的许可证

### 推荐音效类型

**提示音建议：**
- 清脆的铃铛声
- 柔和的电子提示音
- 木鱼声或冥想钟声

**背景音建议：**
- 雨声/海浪声（白噪音）
- 轻柔的钢琴曲
- 森林鸟鸣声
- 咖啡厅环境音

## 音频要求

- 格式：MP3 或 WAV
- 提示音音量：适中（代码中已设置为 50%）
- 背景音音量：较低（代码中已设置为 30%，循环播放）

## 使用方法

1. 从上述网站下载喜欢的音效
2. 重命名为 `notification.mp3` 或 `ambient.mp3`
3. 放入此文件夹（`public/sounds/`）
4. 刷新页面即可生效

## 不添加文件会怎样？

- **提示音**：自动使用内置生成的音效（Web Audio API）
- **背景音**：无法播放，但其他功能正常
